Set Up for macOS
-------

## Install Dropbox

https://www.dropbox.com/install

## Setup github ssh key

## Clone this repository

## Download Terminal Theme (& Restart)
https://draculatheme.com/terminal/

## Install Homebrew
https://brew.sh/

## Create symlink
`./init_mac.sh`

## Install applications
`brew bundle install --global`

## change default shell
`chsh -s /bin/zsh`
